# 🌸 iris flower classifier

my first ever coding project! a little web app that guesses a flower's 
species based on its measurements. dark + moody + lily themed because 
i wanted it to feel like something, not just work.

## try it
🔗 [iris-classifier-deets404.streamlit.app](https://iris-classifier-deets404.streamlit.app/)

## how it works
move the sliders → it guesses if it's a setosa, versicolor, or virginica 
→ shows how confident it is. trained using a random forest model on the 
classic iris dataset.

## made with
python · scikit-learn · streamlit · a lot of custom css

## run it yourself
```bash
git clone https://github.com/deets404/iris-classifier.git
cd iris-classifier
pip install -r requirements.txt
streamlit run app.py
```
